01Google Sheets — каталог и заявки

Шаг 2 — Apps Script

В таблице: Расширения → Apps Script. Вставь код:

const SHEET_ID    = 'ВАШ_ID_ТАБЛИЦЫ';   // из URL таблицы
const TG_BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА';
const TG_CHAT_ID   = 'ВАШ_CHAT_ID';

function doGet() {
  const ss      = SpreadsheetApp.openById(SHEET_ID);
  const sheet   = ss.getSheetByName('Диски');
  const data    = sheet.getDataRange().getValues();
  const headers = data[0];
  const rows    = data.slice(1)
    .filter(r => r[headers.indexOf('active')] === true)
    .map(r => {
      const obj = {};
      headers.forEach((h, i) => obj[h] = r[i]);
      return obj;
    });
  return ContentService
    .createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const data  = JSON.parse(e.postData.contents);
  const ss    = SpreadsheetApp.openById(SHEET_ID);
  const sheet = ss.getSheetByName('Заявки');
  sheet.appendRow([
    new Date(), data.name, data.phone,
    data.tg, data.title, data.platform, data.note
  ]);
  sendToTelegram(data);
  return ContentService
    .createTextOutput(JSON.stringify({ok:true}))
    .setMimeType(ContentService.MimeType.JSON);
}

function sendToTelegram(d) {
  const text = `🎮 ПРЕДЗАКАЗ\n\nИгра: ${d.title} (${d.platform})\nИмя: ${d.name}\nТел: ${d.phone}\nTg: ${d.tg||'—'}`;
  UrlFetchApp.fetch(
    `https://api.telegram.org/bot${TG_BOT_TOKEN}/sendMessage`,
    { method:'POST', contentType:'application/json',
      payload: JSON.stringify({chat_id:TG_CHAT_ID, text}) }
  );
}

02Telegram Bot

Создать бота

  1. Открой Telegram → найди @BotFather
  2. Напиши /newbot → введи имя и username
  3. Скопируй токен вида 1234567890:ABCdef…

  1. Напиши любое сообщение своему боту

03Платформы в блоке Предзаказы

В блоке предзаказов есть фильтр-табы по платформам. Табы генерируются автоматически — какие платформы есть в карточках, те и появляются. Каждой платформе автоматически присваивается цветная полоска.

Список поддерживаемых платформ

Когда заполняешь Google Sheets или JSON-файл в Tina, в колонке platform пиши строго один из этих кодов:

КодПолное названиеЦвет полоски
ps5PlayStation 5Синий градиент
ps4PlayStation 4Тёмно-синий
ps3PlayStation 3Тёмно-синий (приглушённый)
ps2PlayStation 2Тёмный синий
xboxXbox Series X|SЗелёный градиент
xboxoneXbox OneТёмно-зелёный
switchNintendo SwitchКрасный градиент
switch2Nintendo Switch 2Малиновый градиент
pcPC / SteamФиолетовый градиент
Пример: если написать switch2 — на сайте автоматически появится таб «Nintendo Switch 2» и карточки с малиновой полоской. Никакого дополнительного кода не нужно.

Как добавить совсем новую платформу (например PS6)

Нужно прописать её в двух местах в коде сайта:

1. В OLD GAMER.html — найди var PLATFORM_CONFIG и добавь строку:

ps6: { label: 'PlayStation 6', short: 'PS6', bg: 'linear-gradient(90deg,rgba(0,30,120,.9),rgba(0,80,220,.9))' },

2. В nextjs/tina/config.ts — найди PLATFORM_OPTIONS и добавь:

{ value: 'ps6', label: 'PlayStation 6' },

После этого в Google Sheets/Tina пиши ps6 — всё подтянется автоматически.

Как работает автогенерация табов

JS-код на сайте при загрузке читает data-platform каждой карточки и динамически строит фильтр-кнопки. Поэтому:

  • Если нет ни одной карточки с xbox — таб Xbox не появится
  • Как только добавишь хоть одну карточку switch2 — таб Nintendo Switch 2 появится сам
  • Порядок табов: сначала «Все», потом в порядке появления в HTML

В проекте уже подготовлен Next.js стартер (папка nextjs/) с настроенной Tina CMS. После подключения у тебя будет полноценная визуальная админка по адресу твой-сайт.vercel.app/admin/index.html.

Установка и первый запуск

  1. Убедись что установлен Node.js 18+
  2. Открой терминал, перейди в папку: 
    cd nextjs
  3. Установи зависимости: 
    npm install
  4. Скопируй файл окружения: 
    cp .env.local.example .env.local
  5. Запусти в режиме разработки: 
    npm run dev
  6. Открой localhost:3000 — сайт, и localhost:3000/admin/index.html — админка
Локальная разработка: при npm run dev Tina запускается в local-режиме — никакого облака не нужно, всё работает с файлами на диске.

  1. Открой твой-сайт.vercel.app/admin/index.html
  2. Войди через GitHub (тот же аккаунт что и в Tina Cloud)
  3. В левом меню выбери нужную коллекцию (Настройки, Предзаказы и т.д.)
  4. Отредактируй → нажми Save
  5. Tina автоматически делает коммит в GitHub → Vercel пересобирает сайт за 30–60 сек

  1. На app.tina.io → твой проект → Users → Invite
  2. Введи email менеджера — он получит письмо со ссылкой
  3. Менеджер регистрируется через GitHub и получает доступ к админке
Лимиты бесплатного плана Tina: 2 пользователя, неограниченный контент. Vercel Free: 100 GB трафика/мес — этого хватит для магазина с запасом.

05Как вставить карту в блок «Где найти»

Сейчас в блоке «Где найти» стоит заглушка-карта. Есть два способа добавить настоящую карту — Яндекс (рекомендую для России) или Google Maps.

Вариант 1 — Яндекс Карты (рекомендую)

01
Открой конструктор

Зайди на yandex.ru/map-constructor

02
Найди магазин

Введи адрес: «Новосибирск, Троллейная 130а, ТРЦ Континент» → поставь метку на точное место.

03
Настрой вид

Выбери размер (лучше 100% × 100%), отключи лишние элементы, включи метку.

04
Получи iframe

Нажми «Получить код» → скопируй HTML-код с тегом <iframe>.

Открой OLD GAMER.html, найди блок с классом class="map" — он выглядит так:

<div class="map" aria-label="Карта · ТРЦ Континент">
  <!-- Заглушка с меткой -->
  <div class="map-pin">...</div>
</div>

Замени всё содержимое div.map на iframe от Яндекса:

<div class="map" aria-label="Карта · ТРЦ Континент, Новосибирск">
  <iframe
    src="https://yandex.ru/map-widget/v1/?ll=82.9ВАШ_ДОЛГОТА%2C54.9ВАШ_ШИРОТА&z=16&pt=82.9ВАШ..."
    width="100%"
    height="100%"
    frameborder="0"
    style="border:0;border-radius:16px"
    allowfullscreen
    title="OLD GAMER на карте"
  ></iframe>
</div>
Важно: в CSS у .map уже стоит min-height:400px и border-radius:16px. iframe автоматически заполнит весь блок. Атрибут style="border-radius:16px" на самом iframe тоже нужен — браузеры по-разному обрезают углы.

Вариант 2 — Google Maps

  1. Открой maps.google.com
  2. Найди «ТРЦ Континент Новосибирск Троллейная»
  3. Нажми «Поделиться» → «Встроить карту»
  4. Скопируй код <iframe>
  5. Вставь в div.map как показано выше
Google Maps в России может работать медленно или блокироваться — для российской аудитории лучше использовать Яндекс Карты.

Быстрая ссылка (без iframe)

Если не хочешь возиться с iframe — можно просто добавить кнопку «Открыть на карте»:

<a href="https://yandex.ru/maps/-/CDxxxxxx"
   target="_blank"
   class="btn btn-ghost">
  Открыть на Яндекс Картах →
</a>

Ссылку получи в Яндекс Картах: открой место → «Поделиться» → скопируй короткую ссылку.

06Vercel — хостинг без сервера

Vercel — бесплатный хостинг для Next.js. Автодеплой: изменил файл в GitHub → через 30 сек обновилось на сайте.

Обновление сайта без Tina

Если нужно исправить что-то в HTML напрямую:

  1. Открой файл в GitHub → нажми карандаш (Edit)
  2. Внеси правки → Commit changes
  3. Vercel автоматически пересоберёт сайт
Альтернатива без Git: Netlify Drop — просто перетащи папку с файлами, сайт онлайн за 30 секунд. Но тогда Tina CMS не подключить.

07SEO — что уже сделано

  • ✅ <title> с ключевыми словами

  • ✅ JSON-LD LocalBusiness (структурированные данные)

Что нужно сделать вручную

  1. В OLD GAMER.html замени https://oldgamer.ru на реальный URL сайта
  2. Создай sitemap.xml на xml-sitemaps.com
  3. Создай robots.txt в корне:
User-agent: *
Allow: /
Sitemap: https://твойсайт.ru/sitemap.xml
Самое важное для локального SEO: зарегистрируй магазин в Яндекс Бизнес и Google Business Profile. Это даёт появление на картах и в выдаче «магазин видеоигр Новосибирск».

09Яндекс.Вебмастер и Google Search Console

Яндекс.Вебмастер

  1. webmaster.yandex.ru → Добавить сайт
  2. Выбери способ «HTML-файл» → скачай файл yandex_XXXXX.html
  3. Положи его в корень сайта (рядом с OLD GAMER.html)
  4. Нажми «Проверить» → готово
  5. Индексирование → Файлы Sitemap → добавь URL

Google Search Console

  1. search.google.com/search-console → Добавить ресурс
  2. Тип «URL-префикс» → введи URL → подтверди через HTML-тег
  3. Вставь в <head> сайта:
<meta name="google-site-verification" content="ВАШ_КОД">
  1. Нажми «Подтвердить» → добавь Sitemap
