Инструкция для владельца сайта

01Структура сайта

Сайт — статический: обычные файлы HTML/CSS/JS без серверной части. Деплоится на GitHub + Vercel за 5 минут. Диски для предзаказов берутся из Google Sheets через Apps Script Web App.

oldgamer/
├── index.html          ← вся страница сайта
├── styles.css          ← все стили
├── script.js           ← весь JavaScript + интеграция с Sheets
├── privacy.html        ← политика конфиденциальности
├── assets/
│   ├── logo.png        ← логотип в шапке и футере
│   ├── head.png        ← аватарка / фавиконка
│   ├── ps5-hero.png    ← фото консоли в hero-блоке
│   ├── shop-1..4.jpg   ← фото магазина в слайдере
│   └── nintendo-switch.jpg
└── apps-script/
    └── Code.gs         ← код для Google Apps Script

02Google Sheets — каталог предзаказов

Все диски и игры управляются через обычную Google Таблицу. Меняешь строку → обновляешь страницу → карточка на сайте изменилась. Никакого кода не нужно.

Структура листа «Диски»

В таблице должен быть лист с именем Диски. Строка 1 — заголовки (строго как в таблице ниже):

КолонкаНазваниеПримерОписание
Aid1Порядковый номер (любой)
BtitleGTA VIНазвание игры
Cplatformps5Код платформы (см. раздел 04)
DplatformLabelPlayStation 5Полное название платформы
Eprice4 999 ₽Цена строкой
FreleaseРелиз: осень 2026Дата релиза или статус
GmetaAction · Rockstar · SingleЖанр, студия — показывается под названием
Hstatuspreorderpreorder — предзаказ, stock — в наличии
IimageUrlhttps://…/cover.jpgURL обложки. Пусто — авто-арт из названия
JactiveTRUETRUE — показывать, FALSE — скрыть
Подсказка: чтобы быстро заполнить таблицу, попроси ChatGPT:
«Заполни Google Sheets для магазина видеоигр. Колонки: id, title, platform, platformLabel, price, release, meta, status, imageUrl, active. Добавь 15 актуальных игр для PS5, Xbox и Switch на 2025–2026 г. с ценами в рублях. imageUrl оставь пустым.»

Как скрыть/показать игру

В колонке active поставь FALSE — карточка исчезнет с сайта после обновления страницы. Снова TRUE — появится обратно.

Обложки (imageUrl)

  • Если оставить пустым — сайт нарисует стилизованную арт-обложку с названием игры
  • Если вставить URL картинки — будет использоваться она (рекомендуем 600×800 px, JPG)
  • Можно использовать любой публичный URL (Google Drive: файл → «Поделиться → Все» → заменить /file/d/ID/view на /uc?id=ID)

Apps Script — код сервера

В таблице: Расширения → Apps Script. Вставь содержимое файла apps-script/Code.gs из архива проекта. В начале файла укажи ID своей таблицы:

var SPREADSHEET_ID = 'ВСТАВЬТЕ_ID_ТАБЛИЦЫ_СЮДА'; // ← заменить!

ID таблицы — это часть URL между /spreadsheets/d/ и /edit:

https://docs.google.com/spreadsheets/d/ВОТ_ЭТО_И_ЕСТЬ_ID/edit

Telegram-уведомления (опционально)

Чтобы получать уведомления о новых заявках в Telegram, укажи токен бота и chat_id в настройках Apps Script:

  1. В редакторе Apps Script: Настройки проекта (шестерёнка) → Свойства скрипта
  2. Добавь два свойства:
КлючЗначение
TELEGRAM_BOT_TOKENтокен от @BotFather
TELEGRAM_CHAT_IDID чата куда слать уведомления
Если не указать — Apps Script просто пропустит отправку, заявки всё равно сохранятся в лист «Заявки».

03Telegram Bot — уведомления о заявках

Создать бота

  1. Открой Telegram → найди @BotFather
  2. Напиши /newbot → введи имя и username (например oldgamer_orders_bot)
  3. Скопируй токен вида 1234567890:ABCdef…

Узнать Chat ID

  1. Напиши любое сообщение своему боту
  2. Открой в браузере: https://api.telegram.org/botТОКЕН/getUpdates
  3. Найди "chat":{"id":XXXXXXXX} — это твой Chat ID
Совет: создай группу «OLD GAMER — заявки», добавь туда бота и нескольких менеджеров — все будут видеть новые предзаказы одновременно. Chat ID группы тоже берётся через getUpdates.

04Платформы в блоке предзаказов

Фильтр-табы генерируются автоматически — какие платформы есть в активных карточках из Sheets, те и появляются. Каждой платформе автоматически присваивается цветная полоска.

Поддерживаемые коды платформ

В колонке platform в Google Sheets пиши строго один из этих кодов:

КодПолное названиеЦвет полоски
ps5PlayStation 5Синий градиент
ps4PlayStation 4Тёмно-синий
ps3PlayStation 3Тёмно-синий приглушённый
ps2PlayStation 2Тёмный синий
xboxXbox Series X|SЗелёный градиент
xboxoneXbox OneТёмно-зелёный
switchNintendo SwitchКрасный градиент
switch2Nintendo Switch 2Малиновый градиент
pcPC / SteamФиолетовый градиент
Пример: напишешь в Sheets switch2 → на сайте автоматически появится таб «Nintendo Switch 2» и карточки с малиновой полоской. Никакого кода менять не нужно.

Добавить совсем новую платформу

Открой script.js, найди объект PLATFORM_CONFIG в самом начале файла, добавь строку:

ps6: {
  label: 'PlayStation 6',
  short: 'PS6',
  bg:    'linear-gradient(90deg,rgba(0,30,120,.9),rgba(0,80,220,.9))'
},

После этого пиши ps6 в Sheets — цвет и название подтянутся автоматически.

Как работает автогенерация табов

  • Нет ни одной карточки с xbox → таб Xbox не появится
  • Появилась карточка switch2 → таб Nintendo Switch 2 сгенерируется сам
  • Порядок табов: «Все», затем в порядке появления платформ в данных

05GitHub + Vercel — публикация сайта

Сайт — обычные статические файлы. Vercel бесплатно хостит их и автоматически обновляет при каждом изменении в GitHub.

Загрузить на GitHub

Вариант А — через браузер (без Git):

  1. Зарегистрируйся на github.com
  2. New repository → введи название (например oldgamer-site) → Create
  3. «uploading an existing file» → перетащи все файлы проекта → Commit

Вариант Б — через терминал:

git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/ВАШ_НИК/oldgamer-site.git
git push -u origin main

Подключить Vercel

01
Регистрация

Зайди на vercel.com → Sign Up → Continue with GitHub

02
New Project

Add New → Project → импортируй репозиторий oldgamer-site

03
Framework

Framework Preset: Other (статический сайт, не Next.js)

04
Deploy

Нажми Deploy — через 30 секунд сайт онлайн на адресе *.vercel.app

Автообновление: изменил файл в GitHub → Vercel автоматически пересобирает и публикует сайт за 15–30 секунд. Никаких дополнительных действий не нужно.

Свой домен (например oldgamer.ru)

  1. Vercel → твой проект → Settings → Domains → Add
  2. Введи домен: oldgamer.ru
  3. Vercel покажет нужные DNS-записи — добавь их у регистратора домена
  4. Подождать 15–60 минут — домен заработает, SSL-сертификат подключится автоматически

Обновить что-то на сайте без Git

  1. Открой файл в GitHub (например index.html)
  2. Нажми карандаш (Edit this file)
  3. Внеси правки → Commit changes
  4. Vercel автоматически задеплоит изменения
Альтернатива без GitHub: Netlify Drop — просто перетащи папку с файлами, сайт онлайн за 30 секунд. Но тогда автообновление не работает — придётся загружать каждый раз заново.

06Как вставить настоящую карту

Сейчас в блоке «Где найти» стоит стилизованная SVG-заглушка. Когда понадобится настоящая — замени её на iframe от Яндекс Карт.

Вариант 1 — Яндекс Карты (рекомендуем)

01
Конструктор

Зайди на yandex.ru/map-constructor

02
Адрес

Введи «Новосибирск, Троллейная 130а, ТРЦ Континент» → поставь метку.

03
Настройки

Размер: 100%×100%, включи метку.

04
Получи код

«Получить код» → скопируй <iframe>.

Открой index.html, найди <div class="map" id="map-drag" — удали весь SVG внутри и вставь iframe:

<div class="map" aria-label="Карта · ТРЦ Континент, Новосибирск">
  <iframe
    src="https://yandex.ru/map-widget/v1/?ll=82.9XXX%2C54.9XXX&z=16&pt=..."
    width="100%"
    height="100%"
    frameborder="0"
    style="border:0;border-radius:16px;position:absolute;inset:0"
    allowfullscreen
    title="OLD GAMER на карте"
  ></iframe>
</div>
Google Maps в России работает нестабильно — лучше использовать Яндекс Карты.

Кнопка «Открыть на карте» (без iframe)

Если iframe не нужен — добавь кнопку в блок .loc-card:

<a href="https://yandex.ru/maps/-/КОД"
   target="_blank"
   class="btn btn-ghost">
  Открыть на Яндекс Картах →
</a>

Короткую ссылку получи в Яндекс Картах: открой место → «Поделиться».

07SEO — что уже сделано

  • ✅ <title> с ключевыми словами
  • ✅ meta description с адресом и услугами
  • ✅ JSON-LD LocalBusiness (структурированные данные)
  • ✅ Open Graph (превью в соцсетях и мессенджерах)
  • ✅ alt у всех изображений
  • ✅ Семантика H1 / H2 / H3
  • ✅ rel=canonical

Что сделать вручную

  1. В index.html замени все вхождения https://oldgamer.ru на реальный URL
  2. Создай sitemap.xml на xml-sitemaps.com
  3. Создай robots.txt в корне проекта:
User-agent: *
Allow: /
Sitemap: https://твойсайт.ru/sitemap.xml
Самое важное для локального SEO: зарегистрируй магазин в Яндекс Бизнес и Google Business Profile — это даёт карточку магазина в поиске «видеоигры Новосибирск» и на картах.

08Фавиконка

Фавиконка уже подключена — это assets/head.png. Для полного набора (iOS, Android, Windows):

  1. Зайди на realfavicongenerator.net
  2. Загрузи assets/head.png, фон: #0A0D12
  3. Скачай архив → распакуй в assets/favicon/
  4. Замени теги в <head> на предложенные генератором

09Яндекс.Вебмастер и Google Search Console

Яндекс.Вебмастер

  1. webmaster.yandex.ru → Добавить сайт
  2. Способ «HTML-файл» → скачай yandex_XXXXX.html
  3. Положи его в корень проекта (рядом с index.html), запушь в GitHub
  4. Нажми «Проверить» → Индексирование → Файлы Sitemap → добавь URL

Google Search Console

  1. search.google.com/search-console → Добавить ресурс → «URL-префикс»
  2. Подтверди через HTML-тег — вставь в <head> файла index.html:
<meta name="google-site-verification" content="ВАШ_КОД">
  1. Нажми «Подтвердить» → добавь Sitemap
